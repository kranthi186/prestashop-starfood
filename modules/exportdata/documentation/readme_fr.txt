"Exporter les données CSV"

Description:
"Export CSV des données" module vous permet d'exporter les données de Prestashop 1.4 dans Prestashop 1.5.
L'utilisation de ce module ne nécessite pas de compétences techniques particulières, il est facile à utiliser.
Il vous aidera à économiser votre temps de travail de routine de la migration manuelle des données.

Caractéristiques:
Exporter les types de données suivants de Prestashop 1.4 au format CSV en conséquence pour Prestashop 1.5 format CSV d'importation native.:
Une. catégories
2. produits
3. combinaisons
4. clientèle
5. adresses
6. fabricants
7. fournisseurs
Il n'ya aucune limite à la quantité de dossiers!

installation:
Standart Prestashop Module instalation.
